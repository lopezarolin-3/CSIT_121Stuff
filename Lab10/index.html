What is PNG?

PNG (Portable Network Graphics) is a raster image format created as an improvement for GIF (Graphics Interchange Format).

Why it is important

PNG supports lossless compression, transparency, and high-quality images for web use. It is just the best way to add any images to your websites. This is the right tool for specific situations.

Created By

Developed by the W3C to provide a better standard for web graphics.
