Lossless Compression

PNG uses lossless compression, meaning no data is lost when the file is compressed. Also there will be no blurs when the file is compressed.

Transparency

Supports alpha channel transparency for smooth edges and overlays. This is used a lot when people try to add logos or images without any background to their websites.

Color Support

The image is made of a grid of pixels(very tiny dots of color). then the color data is store by each pixel storing color values, usually RGB which is red, green and blue.

Supports millions of colors and grayscale images.

Data Organization

IHDR = Image size, color type.

IDAT = Actual compress image data..

IEND = Marks the end of the file.
